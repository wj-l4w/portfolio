Project information

  • Category: Video Game
  • Project date: July 2021
  • Github link: https://github.com/wj-l4w/OOP_Jun-Space-Shooter

Project Description

One of the first assignments of my degree studies involving an OOP subject, I have decided to make a game using Scala language which I have never done before as all my previous games were developed using game engines.

Skills learnt

As a solo project, I have managed to make a functional game with ScalaFX and SceneBuilder, accompanied by level system, enemy scaling, volume controls and a local database. Enemy logic is a simple script that moves it towards the player within a random range of X and Y positions.

Areas to improve

Compared to my previous games, the simplicity of this game makes it easier to develop, but I still ran into occasional bugs with projectile firing that duplicates the lasers. I also think that the enemy AI could be further improved to accomodate a variety of enemies with different behaviours and attacks.
