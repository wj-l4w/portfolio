Project information

  • Project date: November 2020

Project Description

A continuation in my web-based assignments, this one is related to designing an e-commerce website that includes backend, which marks the first time my team and I have ever touched backend web development.

Skills learnt

We have got more chances to apply our frontend web design skills which includes better CSS while getting to try out PHP, MYSQL and PHPMyAdmin to simulate the backend with AMPPS.

Areas to improve

While our manual CSS styling has improved, we still have not applied responsiveness to this website. Moreover, there are no security measures in place to protect the backend from injection attacks nor encryption used to hash the passwords stored in our database.
